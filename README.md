# Resizer Microservice

Быстрый и надежный микросервис на Go для загрузки, изменения размера (ресайза), обрезки (кропа) и скругления краев изображений "на лету" с поддержкой кэширования и конвертации в современный формат WebP.

## 🚀 Особенности

*   **Скачивание по URL**: Загружает изображение с любого разрешенного домена.
*   **Умный ресайз (Smart Crop)**: Изменяет размер изображения так, чтобы оно полностью заполняло заданную область (`width` и `height`) без искажения пропорций ("сплющивания"), обрезая излишки.
*   **Настройка обрезки**: Параметры `crop_x` (left, center, right) и `crop_y` (top, center, bottom) для якоря обрезки.
*   **Скругление углов**: Параметр `radius` (в процентах от 0 до 100) работает как CSS-свойство `border-radius`, аккуратно скругляя углы вплоть до идеального круга/овала.
*   **Конвертация в WebP**: Поддержка сжатия в формат WebP для экономии трафика (в несколько раз легче PNG).
*   **Управление качеством**: Параметр `q` или `quality` (от 1 до 100) для управления силой визуального сжатия WebP и степени сжатия PNG.
*   **Умное кэширование**: Обработанные изображения сохраняются на диск. Имя файла генерируется на основе всех параметров (размер, радиус, формат, качество). Повторные запросы отдаются моментально с диска.
*   **Безопасность (SSRF/OOM Protection)**: 
    *   Белый список доменов (разрешено качать только с проверенных источников).
    *   Жесткий лимит на размер скачиваемого файла (максимум 15 МБ), защита от атак исчерпания памяти.
    *   Сетевые таймауты (15 секунд) для предотвращения зависания горутин.
*   **Graceful Shutdown**: Безопасная остановка сервера с ожиданием завершения текущих обработок (до 10 секунд).
*   **Метрики в заголовках**: Возвращает `X-Download-Time`, `X-Processing-Time` и уникальный `X-Image-Hash` (SHA-256) в HTTP заголовках.

## 🛠 Установка и запуск

### Зависимости WebP (Опционально)

Для работы с форматом WebP используется гибридная библиотека `gen2brain/webp`. 

**Она может работать двумя способами:**
1. **Максимально быстро (через системные библиотеки)**. Если в системе установлена C-библиотека `libwebp`, Go-сервис автоматически найдет её и подключится (без необходимости CGO). Рекомендуется для продакшена:
   - macOS: `brew install webp pkg-config`
   - Linux: `sudo apt-get install libwebp-dev pkg-config`
2. **"Pure Go" Fallback (без внешних зависимостей)**. Если сис. библиотек нет (или вы собираете Docker образ "с нуля" без зависимостей через `CGO_ENABLED=0`), сервис автоматически переключается на встроенный WASM-движок и выполняет кодирование **силами самого Go**. Это работает везде "искаробки" без установок, но процесс ресайза будет происходить чуть медленнее.

### Запуск сервера

```bash
# Установка Go-зависимостей
go mod tidy

# Запуск
go run main.go
```

Сервер запустится на порту `8085` (настраивается).

## ⚙️ Конфигурация

Конфигурация происходит через файл `config.yml` в корне проекта или через переменные окружения.

Пример `config.yml`:
```yaml
server:
  port: 8085
security:
  allowed_domains:
    - s1.1zoom.ru
    - get.wallhere.com
    - img.freepik.com
storage:
  path: "artefacts" # Папка для кэширования файлов
```

Переменные окружения имеют приоритет над файлом `config.yml`:
*   `PORT=8080`
*   `ALLOWED_DOMAINS=domain1.com,domain2.com`
*   `STORAGE_PATH=/tmp/cache`

## 📖 API Usage (Использование)

Формат запроса:
`GET http://localhost:8085/?url={IMAGE_URL}&width={W}&height={H}&radius={R}&crop_x={CX}&crop_y={CY}&format={F}&q={Q}`

### Параметры (`Query string`):

| Параметр | Описание | Значение по умолчанию | Примеры |
| :--- | :--- | :--- | :--- |
| **`url`** | Ссылка на оригинальное изображение (обязательно). Домен должен быть в белом списке. | - | `https://example.com/img.jpg` |
| **`width`** | Желаемая ширина изображения в пикселях. | Оригинал | `300` |
| **`height`** | Желаемая высота изображения в пикселях. | Оригинал | `300` |
| **`crop_x`** | Якорь обрезки по оси X. | `center` | `left`, `center`, `right` |
| **`crop_y`** | Якорь обрезки по оси Y. | `center` | `top`, `center`, `bottom` |
| **`radius`** | Радиус скругления углов в **процентах** от максимально возможного. | `0` (без скруглений) | `0` ... `100` (100 = круг/овал) |
| **`format`** | Формат отдаваемого изображения. | `png` | `png`, `webp` |
| **`q`** (или `quality`) | Качество/Сила сжатия (от 1 до 100). | `80` | `50` (сильное сжатие для webp) |

### Примеры запросов:

**Простой ресайз:**
```
http://localhost:8085/?width=300&height=200&url=https://s1.1zoom.ru/big7...
```

**Ресайз + Смещение обрезки (кроп по верхнему левому краю):**
```
http://localhost:8085/?width=300&height=300&crop_x=left&crop_y=top&url=...
```

**Превращение в круг (radius=100) + Формат WebP + Качество 60:**
```
http://localhost:8085/?width=400&height=400&radius=100&format=webp&q=60&url=...
```
